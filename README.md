# 자료구조와 알고리즘 with Java

## 개요 📄

이 저장소는 책 `'자바 알고리즘 인터뷰'`의 내용을 기반으로 **자료구조와 알고리즘**을 학습한 내용이 담겨있습니다. \

뿐만 아니라, 백준 온라인 저지의 다양한 문제에 도전해보았습니다. \
이를 통해 이론을 실제로 적용하는데에 중점을 두었고, 책 외의 다양한 문제를 풀이하였습니다.
